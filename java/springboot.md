Gradle 和 Maven

1. 配置

maven 用 pom.xml (project Object Model)
gradle 用 build.gradle

2. 灵活性

gradle 更灵活, 自定义流程和构建过程 , maven 需要通过插件来实现

3. 插件生态

都不错

4. 性能

大项目时候 gradle 比 maven 快

5. 学习曲线

maven 比 gradle 简单, gradle提供更多灵活性和可能，也更复杂
